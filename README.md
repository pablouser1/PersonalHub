# Personal Hub
My personal page made using Astro, Svelte and Bulma

## Installation
```bash
yarn install
```

## Deploy
```bash
yarn build
```

## Config
### astro.config.mjs
Open the file and change the "site" value

### .env
Copy `.env.sample` to `.env` and modify however you like

### .json
Copy all the `/src/data/*.sample.json` files and modify them
